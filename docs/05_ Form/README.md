### :white_check_mark: 주요 내용

- 코드 형식은 의사소통의 일환이다.
- 버전이 바뀌어도 초기의 스타일과 가독성 수준은 유지보수 용이성과 확장성에 영향을 미친다.
- 적절한 자바 소스 파일의 크기 : 평균 65줄
- 적절한 행 길이 : 평균 45자, 120자 제한
