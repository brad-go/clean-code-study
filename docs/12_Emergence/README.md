### 창발성

- 자기 조직화 현상
- 하위 계층에는 없는 특성이나 행동이 상위 계층에서 자발적으로 돌연히 출연하는 현상

### 켄트 벡이 제시한 단순한 설계 규칙 중요도 순 4가지

1. 모든 테스트를 실행한다.
2. 중복을 없앤다.
3. 프로그래머 의도를 표현한다.
4. 클래스와 메서드 수를 최소로 줄인다.
