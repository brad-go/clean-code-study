### 주요 내용

깨끗한 코드를 작성하기 위해서는 **장인정신**이 필수적이다.

- 원칙, 패턴, 기법, 경험이라는 지식을 습득한다. (이론)
- 열심히 일하고 연습해 지식을 몸과 마음으로 체득한다. (실전)
