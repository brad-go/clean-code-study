### :white_check_mark: 주요 내용

- 프로그래머는 어떤 식으로든 외부코드(패키지, 오픈소스, 다른 부서가 제공하는 컴포넌트)를 사용

- 경계에 위치하는 코드는 깔끔히 분리

- 통제가 가능한 우리 코드에 의존하여 안에 숨김

- 패키지를 호출하는 코드를 줄임

- 추후 패키지가 변했을 때 변경할 코드가 적도록 작성
